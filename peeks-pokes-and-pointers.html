ğŸŒŸ Common Gotchas

Block vs Inline Elements

Know the difference! Some elements are block-level (<div>, <p>, <h1>) and some are inline (<span>, <a>, <em>).

The Box Model

Remember: width/height + padding + border = total size (unless you use box-sizing: border-box)

Z-Index Only Works on Positioned Elements

Need position: relative/absolute/fixed for z-index to work!

Margins Collapse

Vertical margins between elements collapse to the larger of the two, not their sum.
